# 极光栈（AuroraHub）官网开发规范

## 1. 文件
1. 源文件：
1.1. 源文件存放在 `/` 目录下。
1.2. 源文件命名规则：
1.2.1. 源文件名：`<网页名称>`
1.3. js和css的存放位置
1.3.1. js和css的存放位置：`/src`
1.3.2. js的文件名：`script-<网页名称>`
1.3.3 css的文件名：`style-<网页名称>`
1.4. 图片的存放位置：`/img`
1.5. 文档的存放位置：`/docs`

## 翻译（i18n国际化）
- 翻译文件存放在 `/` 目录下。
- 当你每次新增了按钮/输入框等文字时，请在翻译文件中分别增加zn和en
